Data Cleaning and Standardisation in SQL

This project involved cleaning and standardising data in an SQL database for a Nashville housing dataset. It also involved changing categorical values (e.g., 'Y' and 'N' to 'Yes' and 'No'), removing duplicates using row numbering, and deleting unused columns.
